# <img src="https://cdn-icons-png.flaticon.com/128/11315/11315900.png" width="20" /> Credit Scorecard Development
Credit scorecard development is a methodological approach to quantify a borrower's willingness and ability to repay. Credit scorecards have significant impact on lending decisions and loan processes, particularly for banks and fintech companies. This repository provides a detailed walkthrough of the entire credit scorecard development process, explaining each step.
