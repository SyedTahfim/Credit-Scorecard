# <img src="https://cdn-icons-png.flaticon.com/128/11315/11315900.png" width="20" /> Credit Scorecard Model
The credit score model is developed to enable objective, data-driven assessment of borrower creditworthiness and reduce subjectivity in lending decisions. It helps identify and quantify default risk, supporting faster and more consistent credit approval processes. The model also enhances portfolio quality by improving risk-based customer segmentation and minimizing potential credit losses.
